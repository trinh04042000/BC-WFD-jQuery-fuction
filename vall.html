Hàm vall

Ý nghĩa

Phương thức val () trả về hoặc đặt thuộc tính giá trị của các phần tử được chọn.

Cú pháp

Trả về thuộc tính giá trị

$(selector).val()

Set the value attribute:

$(selector).val(value)

Set the value attribute using a function:

$(selector).val(function(index,currentvalue))
